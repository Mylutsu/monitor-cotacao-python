# 🪙 Monitor de Cotação de Moedas

Projeto simples em Python que consome uma API de economia para exibir valores atualizados de Dólar, Euro e Bitcoin.

## 🛠️ Tecnologias Utilizadas
* **Python 3**
* **Biblioteca Requests** (para integração com APIs)
* **JSON** (formato de tratamento de dados)

## 📖 O que eu aprendi
Neste projeto, pratiquei como realizar requisições HTTP, como manipular dicionários no Python e como formatar saídas numéricas.
